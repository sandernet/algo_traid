Создание виртуальной среды
python3 -m venv <имя среды>
python -m venv venv

Запуск linux
source venv/bin/activate
Запуск Windows
venv\Scripts\activate


Файл Requirements
pip freeze > requirements.txt
pip install -r requirements.txt
